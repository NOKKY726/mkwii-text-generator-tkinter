■ Description
Characters that can be used are  A to Z  1 to 9  :  .  +  -  /
The second fonts are available for 1-9 and - /
It can be used by enclosing it in single quotation (').
Space enclosed with ' is one-sixth length of normal space.

Example)
LAP '1   '  00:23.018
LAP '2'  00:22.884
LAP '3'  00:22.872
